## วิธีติดตั้ง

1. แตกไฟล์ ZIP
2. เลือกไฟล์และโฟลเดอร์ทั้งหมดภายใน
3. อัปโหลดทับไฟล์ใน GitHub Repository `clob`
4. Commit changes
5. รอ Vercel Deploy
6. หากโลโก้เดิมยังแสดง ให้ Refresh แบบล้าง Cache หรือเปิดเว็บใหม่ใน Incognito
