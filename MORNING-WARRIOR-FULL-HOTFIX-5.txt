Install:
1. Back up the current public folder.
2. Replace the entire public root with all files and folders from this ZIP.
3. Publish/deploy.
4. Open the production PWA. It can reload itself once while Hotfix 5 takes
   control; this is expected.
5. Confirm the version label shows Hotfix 5.
6. Test one AI food-photo analysis on the production domain.

Firebase data remains under the existing `clob` paths. This build does not
