CLOB Patch-002 — Exercise Library Protection

วางทับไฟล์เดิม 3 ไฟล์:
- js/exercise-library.js
- js/exercise-library-page.js
- sw.js

สิ่งที่แก้:
1. Core Library (22 ท่าพื้นฐาน) ถูกฝังในแอปและจะไม่ถูก Firebase/localStorage เขียนทับอีก
2. ท่าที่ Trainer เพิ่มเองเก็บแยกเป็น Custom Library
3. หน้า Library และตัวเลือกท่าใน Program จะแสดง Core + Custom รวมกัน
4. Core exercise เป็น read-only: แก้ไขหรือลบไม่ได้
5. Custom exercise เพิ่ม/แก้ไข/ลบได้ตามเดิม
6. กู้ Custom exercise จาก cache เก่าโดยอัตโนมัติ และตัด built-in records ที่ซ้ำออก
7. เปลี่ยน Service Worker cache เพื่อบังคับโหลดไฟล์เวอร์ชันใหม่

หลังอัปโหลด:
- เปิดเว็บใน Private Tab หรือปิดแล้วเปิด Safari ใหม่
- เข้า Trainer > Library
- ควรเห็น Core exercises กลับมาพร้อม Side Plank ที่สร้างเอง
- ลองเพิ่มท่าใหม่อีก 1 ท่า แล้ว Refresh; ทั้ง Core และ Custom ต้องยังอยู่
